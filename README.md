# 光盘存档工具

将一个仍会更新目录中的文件，按从旧到新的时间顺序，分装到多个光盘中，实现备份功能。

## 特点

- 将目录中的文件平铺到光盘中

- 按从旧到新的时间顺序排序，而不是按目录，这可以让新的数据无需插入到旧的光盘中

- 在备份时平铺所有文件，恢复时能够重建目录结构

- 支持根据时间，备份任意修改时刻后的文件，实现接续导出
